# Personal Portfolio Website

This repository contains my personal portfolio website, built to showcase my projects, skills, and professional journey as a Computer Science student.

## About
The website presents:
- A brief introduction about me
- Selected academic and personal projects
- A timeline of my education and experience
- A downloadable resume

## Purpose
This project was created as part of my personal portfolio to support job applications and demonstrate my approach to building clean, well-structured software projects.

## Status
The website is actively developed and continuously improved as I gain more experience and complete new projects.
